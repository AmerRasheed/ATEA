Here is a simple automation test case. 

The user go to the site "http://app.matglad.nu/#/startview" and click on the "Inställningar" button. On clicking it, the control is transferred (from main page) to a new webpage "Mina Inställningar". 

Finally, the RETURN button is pressed and control is transferred back to the main page again.


Installation required: 
1-Python
2-PyCharm IDE
3-RobotFramework
4-RobotFramework+Selenium Libraries

PyCharm Packages
Selenium, RobotFramework, and RobotFramework+Selenium.

PyCharm Plugins
Intellibot@Selenium Library

Finally, open Robot Framework Project and run the Test.robot file by using the command robot Test.robot.
